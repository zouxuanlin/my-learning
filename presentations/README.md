## 使用说明

演示文稿采用 Markdown 格式编写，便于版本控制和协作。每个演示文稿应包含：
- 清晰的标题和副标题
- 目录结构（如内容较多）
- 图表和代码示例（如适用）
- 参考资料和引用

## 创建新演示文稿

1. 在本目录下创建新的 `.md` 文件
2. 使用一致的格式和样式
3. 在本 README 中添加链接和简要描述
4. 提交到 Git 仓库
