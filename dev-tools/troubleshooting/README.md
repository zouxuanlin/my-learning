# 问题解决方案与故障排除

本目录收集各类技术问题的解决方案、故障排除指南和调试技巧。

## 文档分类

- **Git相关问题**：版本控制工具的常见问题和解决方案
- **开发环境问题**：终端、编辑器、系统配置等开发环境问题
- **MCP/Agent问题**：Claude Code、OpenClaw等智能体相关的问题解决方案
- **网络和连接问题**：API连接、代理配置、网络超时等问题
- **软件安装问题**：各类软件和框架的安装、配置问题

## 使用建议

- 遇到问题时，先在此目录搜索相关关键词
- 解决新问题后，请及时记录并添加到此目录
- 每个解决方案文档应包含：问题描述、错误信息、根本原因、解决方案步骤
