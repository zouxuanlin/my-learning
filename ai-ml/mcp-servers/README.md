# MCP 服务器文档与配置

本目录包含各种MCP（Model Context Protocol）服务器的相关文档、配置指南和使用说明。

## 目录内容

- **Browser MCP**: 浏览器自动化MCP服务器的完整指南和配置文档
- **GitHub MCP**: GitHub集成MCP服务器的文档和故障排除指南  
- **MCP服务器分类**: Awesome-MCP-ZH项目整理的533个MCP服务器分类速查表
- **通用指南**: MCP服务器的组织、配置和最佳实践

## 用途

这些文档帮助开发者：
- 了解和选择合适的MCP服务器
- 正确配置和集成MCP服务器
- 解决常见的MCP连接和配置问题
- 掌握MCP服务器的最佳实践
