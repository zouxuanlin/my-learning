# AI 工具使用指南

本目录包含各种AI工具的使用指南、功能介绍和最佳实践。

## 包含的工具

- **NotebookLM**: Google的AI笔记工具，用于知识管理和内容生成
- **DeerFlow**: 字节跳动开源的超级智能体框架
- **Step3.5 Flash**: 阶跃星辰的开源大语言模型

## 文档结构

每个工具都有独立的Markdown文档，包含：
- 工具功能和架构介绍
- 安装和配置指南
- 使用技巧和最佳实践
- 常见问题和解决方案

## 贡献指南

添加新的AI工具文档时，请遵循以下格式：
1. 创建以工具名称命名的Markdown文件（如 `tool-name.md`）
2. 包含完整的功能介绍和使用示例
3. 确保不包含任何敏感信息或API密钥
4. 更新本README中的工具列表
