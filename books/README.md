# 书籍笔记和摘要

本目录用于存储各类技术书籍、专业书籍的阅读笔记、摘要和关键知识点整理。

## 目录结构

- 每本书籍建议创建独立的子目录或Markdown文件
- 文件命名建议：`书名-作者.md` 或 `书名-作者/`
- 包含书籍的核心概念、重要观点、实践应用等内容

## 内容规范

- **书籍信息**：书名、作者、出版信息、ISBN等
- **阅读进度**：当前阅读章节和完成度
- **核心摘要**：书籍的主要观点和框架
- **关键知识点**：重要的概念、方法、技巧
- **个人思考**：结合实际的思考和应用场景
- **行动项**：从书中获得的可执行建议

## 适用书籍类型

- 技术类书籍（编程、系统设计、AI/ML等）
- 计算机科学经典著作
- 商业和技术管理书籍
- 个人成长和学习方法类书籍

## 注意事项

- 避免直接复制大段原文，注重理解和重构
- 保持笔记的实用性和可检索性
- 定期回顾和更新笔记内容
