## 📚 官方核心资源

### 1. The Rust Programming Language (官方书籍)
- **链接**: https://doc.rust-lang.org/book/
- **中文名**: 《Rust 程序设计语言》
- **特点**: 官方权威教程，覆盖 Rust 所有核心概念
- **适合**: 初学者必读

### 2. Rust by Example (示例学习)
- **链接**: https://doc.rust-lang.org/rust-by-example/
- **特点**: 通过实例学习 Rust 语法
- **适合**: 动手实践型学习者

### 3. Rustlings (练习项目)
- **链接**: https://github.com/rust-lang/rustlings
- **特点**: 小型练习项目，修复错误代码学习 Rust
- **适合**: 初学者入门练习

### 4. Rust Reference (参考手册)
- **链接**: https://doc.rust-lang.org/reference/
- **特点**: 详细的语言规范参考
- **适合**: 进阶查阅

## 🎯 学习路线

### 阶段一：入门 (1-2 周)
1. 安装 Rust (rustup)
2. 完成 Rustlings 前 20 个练习
3. 阅读《The Rust Book》第 1-6 章
4. 理解核心概念：
   - 所有权 (Ownership)
   - 借用 (Borrowing)
   - 生命周期 (Lifetimes)

### 阶段二：基础 (2-4 周)
1. 完成 Rustlings 全部练习
2. 阅读《The Rust Book》剩余章节
3. 学习标准库常用模块
4. 编写小型命令行工具

### 阶段三：进阶 (1-2 月)
1. 学习异步编程 (async/await)
2. 理解 Rust 内存模型
3. 学习常用 crate：
   - serde (序列化)
   - tokio (异步运行时)
   - anyhow/thiserror (错误处理)
4. 参与开源项目

### 阶段四：精通 (持续)
1. 阅读优秀开源项目源码
2. 学习 unsafe Rust
3. 性能优化
4. 贡献社区

## 💡 学习建议

1. **不要跳过所有权**：这是 Rust 最核心也最难的概念
2. **多写代码**：Rust 编译器是最好的老师
3. **理解错误信息**：Rust 的错误提示非常友好
4. **加入社区**：遇到问题及时提问
5. **循序渐进**：不要急于求成，Rust 学习曲线较陡
